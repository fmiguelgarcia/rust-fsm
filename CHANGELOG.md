## [Unreleased]

### [0.1.0] - 2025-12-22

### Added

- Forked from [https://github.com/eugene-babichenko/rust-fsm]
- **Transition guards** using `if` or `match`.
- Input type supports **tuple variants** like `Input::X(u32, String)`.
- Output can be generated by a **closure expression**.
- New **before and after transition hooks** added to `StateMachine`
  and `state_machine` macro
